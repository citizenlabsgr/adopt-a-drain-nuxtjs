# Scripts
These scripts are handy.

## Docker
UP


```
cd adopt-a-drain-nuxtjs/scripts

./dk.up.sh

```

DOWN

```
cd adopt-a-drain-nuxtjs/scripts

./dk.down.sh
```

Clean out old docker images

```
cd adopt-a-drain-nuxtjs/scripts

./dk.prune.sh
```

## Tests
Runs inside the docker container

```
cd adopt-a-drain-nuxtjs/scripts
./dk.tests.sh
```
